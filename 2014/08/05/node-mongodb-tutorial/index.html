10.一些坑

在将数据库和工程文件夹data相连接时，我们通常会在mongo安装目录下执行mongod --dbpath "f://rs//nodetest1//data"或者在mongo安装目录下执行mongo命令，它有时会报错“errno:10061”。

原因：工程文件夹data下存在了mongod.lock文件。这个文件阻止其他进程使用此工程目录。
解决办法：删除工程文件夹下的data文件夹里的mongod.lock文件。
