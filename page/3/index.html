git 命令大全

  • git init
    在项目根目录执行git init,根目录会多出一个.git的文件夹，这个文件夹用于保存如何git相关的操作记录。
  • git clone
    cd进入项目文件夹，执行git clone将会复制别人提交的版本或者你刚在github/gitlab上创建的项目
